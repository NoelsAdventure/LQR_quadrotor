### LQR Quadrotor

This repository presents a cascaded Linear Quadratic Regulator (LQR) framework for trajectory tracking on a non-linear quadrotor modeled with a +–frame motor configuration. The controller consists of an outer-loop translational LQR and an inner-loop attitude LQR. A full simulation environment is developed using physically based inertial and thrust parameters, and the controller is evaluated on circular, helical, and lemniscate trajectories. The results show that the cascaded LQR achieves stable tracking, maintains bounded errors, and generates feasible motor forces across all scenarios. These findings demonstrate that cascaded LQR provides a simple, computationally efficient baseline for quadrotor control and offers a strong foundation for future safety-critical extensions.
